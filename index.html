Web Proxy Viewer

🍕 HTML取り寄せ屋

他サイトのURLを入れると、そのHTMLを取得してここに表示します。

ヒント: このファイルを file://から直接開いていませんか?その場合、ブラウザの制限で取得が失敗しやすくなります(特にSafariで顕著です)。ターミナルでこのフォルダに移動して python3 -m http.server 8000 を実行し、http://localhost:8000/ からアクセスすると改善することがあります。

Order — 取得するURL

URLを入力して「取得する」を押してください。

上のURL欄にアドレスを入力して「取得する」を押すと、ここにプレビューが表示されます。

※ このツールは無料の公開CORSプロキシ(allorigins.win → codetabs.com → corsproxy.io の順)を経由して他サイトのHTMLを取得しています。1つ失敗しても自動で次を試します。ログインが必要なページ、フレーム埋め込みを制限しているサイト、厳格なセキュリティポリシー(CSP)を持つサイト、Googleのようにボット対策が強いサイトは正しく表示されない場合があります。取得先サイトの利用規約の範囲でご利用ください。
